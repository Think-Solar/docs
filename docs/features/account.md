# Account

This will display the authenticated users account details.

Users can edit various account details based on their assigned [Role](/features/roles).

## Features

### Displayed details

- Name
- Role
- Email
- Phone number
- address
- Licence number
- Signature
- Company documents
- Health & safety documents

### Admin only displayed details

 - Labour details
    - Labour cost price
    - Labour sell price
